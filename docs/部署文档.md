# 部署文档

## 环境要求
- JDK 17+
- MySQL 8+
- Redis（可选）
- MinIO（可选）

## 部署步骤
1. 配置环境变量（DB/MINIO/JWT/RBAC）。
2. 初始化数据库 `backend/src/main/resources/db/schema.sql`。
3. 打包：`cd backend && mvn clean package -DskipTests`。
4. 启动：`java -jar target/excel-process-1.0.1.jar --spring.profiles.active=prd`。

## 运行后验证
- 健康检查：`/actuator/health`
- Swagger：`/swagger-ui/index.html`
